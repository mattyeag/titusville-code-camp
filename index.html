Welcome to Titusville code-camp

This hands-on code-camp teaches students how the web works, how to write code, and how to build real websites from scratch. No experience required — just curiosity and a willingness to learn.

Code-camp Modules

  • Module 1: Intro to computing, the internet, and how websites work.
  • Module 2: Programming basics using JavaScript — functions, loops, logic.
  • Module 3: Algorithmic thinking and problem solving.
  • Module 4: HTML fundamentals and semantic structure.
  • Module 5: CSS styling, layout, and responsive design.
  • Module 6: DOM manipulation and interactive web pages.
  • Module 7: Git, GitHub, clean code, and deploying projects.

By the end of the code-camp, students will have a deployed portfolio site and real foundational skills.

Who's Teaching the Course

Matthew Yeager

I didn’t start out in tech, but once I tried coding, I was hooked. I loved the mix of problem-solving, creativity, and turning ideas into real projects. Since then, I’ve spent the last 10 years working as a professional software engineer, building modern web applications, large systems, and AI-powered tools. This code camp exists to give people access to those same skills, whether you are technically curious, unsure of your next step, trying to promote a small business, or simply want to build something cool. The course is hands-on, encouraging, and focused on learning by doing. My mindset is simple. If I can learn this, you can too, and with today’s tools, a little knowledge can unlock some pretty amazing possibilities.
