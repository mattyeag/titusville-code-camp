Welcome to Titusville code-camp

This hands-on code-camp teaches students how the web works, how to write code, and how to build real websites from scratch. No experience required — just curiosity and a willingness to learn.

Ideal for beginners who are curious about technology and ready to build practical, career-relevant skills.

Anticipated Schedule & Format

  • Format: 5-6 week course
  • Frequency: One class per week
  • Session Length: Approximately 2 hours
  • Planned Timeline: Spring 2026

The course is designed to allow time between sessions for practice and course projects. Final dates and exact timing will be determined based on community interest and availability.

What You’ll Learn

Titusville Code Camp begins with a structured, beginner-friendly web development course designed to take you from zero experience to building and deploying a real website.

Web Development I: Foundations

Beginner · No Experience Required · Multi-Class Course

Learn how the web works and build your first real website from scratch.

  • How the internet and websites function
  • Writing clean, semantic HTML
  • Styling with CSS and responsive layouts
  • Introductory JavaScript for basic interactivity
  • Project structure, Git, and deployment

Outcome: A fully deployed personal website you can share.

Coming Soon: Web Development II and Software Engineering, will expand into advanced JavaScript, dynamic interfaces, and working with real-world data. Scheduling details will be announced in the coming weeks.

Code Camp Modules

Course 1: Web Foundations

  • Module 1: Intro to computing, the internet, and how websites work
  • Module 4: HTML fundamentals and semantic structure
  • Module 5: CSS styling, layout, and responsive design
  • Module 6 (Intro): Basic JavaScript and simple interactivity
  • Module 7: Git, GitHub, clean code, and deployment

By the end of the code camp, students will have a deployed portfolio site and a strong foundation for further learning or real-world projects.

What You’ll Need

Getting started with Titusville Code Camp is simple. You don’t need prior experience or expensive equipment — just a few basics and a willingness to learn.

  • A Laptop Computer
    Mac or Windows is perfectly fine. As long as it can run a modern web browser (like Chrome) and a code editor, you’re good to go.
  • Google Chrome
    We’ll use Chrome for learning, testing, and debugging our websites. (It’s free and easy to install.)
  • A Free Code Editor (Visual Studio Code)
    We’ll guide you step-by-step in setting this up. No complicated tools or paid software required.
  • $25 Course Fee (Due at first class)
    Covers instruction, materials, and support throughout the course.
  • Curiosity & a Ready-to-Learn Attitude
    This matters more than experience. If you enjoy problem-solving, creativity, or figuring things out, you’ll fit right in.

No experience required.
We start from the very beginning and build together — asking questions is encouraged, and mistakes are part of the process.
