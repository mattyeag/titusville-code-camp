Welcome to Titusville code-camp

This hands-on, in-person bootcamp teaches students how the web works, how to write code, and how to build real websites from scratch. No experience required — just curiosity and a willingness to learn.

Ideal for anyone! Especially beginners who are curious about technology, business owners who want to build digital presence, and anyone ready to build practical, career-relevant skills. If you are interested in pursuing a career in software engineering, this course is a great starting point.

The course will be fast paced and students will be expected to practice and work on projects in between classes.

When and Where

  • Time: Tuesday evenings 6:30pm - 8:30pm
  • Date: April 21st - May 19th
  • Sessions: 5 sessions (potential 6th session on May 26th)
  • Location: Titusville

The course is designed to allow time between sessions for practice and course projects. A workshop will be offered on most Saturdays at Iron Furnace Coffee in Franklin for students who want help with their projects.

What You’ll Learn

Titusville Code Camp begins with a structured, beginner-friendly web development course designed to take you from zero experience to building and deploying a real website. Each session will build on the last with hands-on learning. Practice exercises will be provided each week to apply what you’ve learned. Starting week 2 students will have a project provided which will result in thier completed personal website by the end of the course.

Web Development I: Foundations

  • Week 1: The internet, Software, and how websites function
  • Week 2: Writing clean, semantic HTML. UX design and layouts
  • Week 3: HTML continued and mobile layouts styling with CSS
  • Week 4: Introductory JavaScript for basic interactivity
  • Week 5: Project structure, Git, GitHub and deployment

Outcome: A fully deployed personal website you can share.

Coming Soon: Web Development II and Software Engineering, will expand into advanced JavaScript, dynamic interfaces, and working with real-world data. Scheduling details will be announced in the coming weeks.

What You’ll Need

Getting started with Titusville Code Camp is simple. You don’t need prior experience or fancy equipment — just a few basics.

  • A Laptop Computer
    Mac or Windows is perfectly fine. As long as it can run a modern web browser (like Chrome) and create and save files you’re good to go. (A chromebook needs to have the Linux development environment option to work for this course)
  • Google Chrome
    We’ll use Chrome for learning, testing, and debugging our websites. (It’s free and easy to install.)
  • A Free Code Editor (Visual Studio Code)
    We’ll guide you step-by-step in setting this up. No complicated tools or paid software required.
  • $25 Course Fee (Due at first class)
    Covers instruction, materials, and support throughout the course.
  • Curiosity & a Ready-to-Learn Attitude
    This matters more than experience. If you enjoy problem-solving, creativity, or figuring things out, you’ll fit right in.

No experience required.
We start from the very beginning and build together — asking questions is encouraged, and mistakes are part of the process.
