Welcome to Titusville code-camp

This hands-on code-camp teaches students how the web works, how to write code, and how to build real websites from scratch. No experience required — just curiosity and a willingness to learn.

What You’ll Learn

Titusville Code Camp is split into two multi-class courses. Each course builds on the last, taking students from zero experience to building real, interactive websites.

Course 1: Web Foundations

Beginner · Multiple Classes · No Experience Required

Learn how the web works and build your first real website using HTML, CSS, and introductory JavaScript.

  • How the internet and websites work
  • Writing clean HTML and semantic structure
  • Styling pages with CSS and responsive layouts
  • Intro to JavaScript for basic interactivity
  • Project structure, Git, and deployment

Outcome: A fully styled personal website you can share.

Course 2: Dynamic Web Development

Intermediate · Multiple Classes · Course 1 or equivalent required

Go deeper into JavaScript to create interactive, data-driven websites that respond to users and external data.

  • Advanced JavaScript and DOM manipulation
  • Building dynamic UI features
  • Fetching data from APIs

Outcome: Interactive projects and a deployed portfolio.

What You’ll Need

Getting started with Titusville Code Camp is simple. You don’t need prior experience or expensive equipment — just a few basics and a willingness to learn.

  • A Laptop Computer
    Mac or Windows is perfectly fine. As long as it can run a modern web browser (like Chrome) and a code editor, you’re good to go.
  • Google Chrome
    We’ll use Chrome for learning, testing, and debugging our websites. (It’s free and easy to install.)
  • A Free Code Editor (Visual Studio Code)
    We’ll guide you step-by-step in setting this up. No complicated tools or paid software required.
  • $25 Course Fee (Course 1)
    Covers instruction, materials, and support throughout the course.
  • Curiosity & a Ready-to-Learn Attitude
    This matters more than experience. If you enjoy problem-solving, creativity, or figuring things out, you’ll fit right in.

No experience required.
We start from the very beginning and build together — asking questions is encouraged, and mistakes are part of the process.
